# 🎯 Google Sheets & Forms 자동 채점 시스템

Google Sheets와 Google Forms를 연동하여 자동으로 심사 점수를 계산하고 관리하는 시스템입니다.

## 📋 주요 기능

1. **참가자 관리**: 미리 작성된 명단 기반으로 설문 완료 여부 추적
2. **Google Forms 자동 생성**: 참가자 명단을 기반으로 설문지를 자동으로 생성
3. **실시간 모니터링**: 관리자가 실시간으로 설문 완료 현황 확인
4. **공정한 점수 계산**: 
   - 본인 팀 제외 평가
   - 11개 항목별 채점 (각 1-10점, 총 100점 만점)
   - MVP 점수 별도 계산
   - 팀원 수로 나눈 평균 점수 계산
5. **대시보드**: 실시간 심사 현황 및 최종 점수 확인 (MVP 점수 포함)

## 🚀 설치 및 설정 방법

### 1단계: Google Sheets 생성

1. Google Drive에서 새 Google Sheets 생성
2. 시트 이름을 원하는 이름으로 변경 (예: "자동채점시스템")

### 2단계: Google Apps Script 설정

1. Google Sheets에서 `확장 프로그램` > `Apps Script` 클릭
2. `Code.gs` 파일의 내용을 복사하여 붙여넣기
3. `index.html` 파일 생성:
   - 왼쪽 상단 `+` 버튼 클릭 > `HTML` 선택
   - 파일 이름을 `index`로 변경
   - `index.html` 파일의 내용을 복사하여 붙여넣기

### 3단계: 스크립트 ID 설정

1. `Code.gs` 파일에서 다음 부분을 수정:
   ```javascript
   const SPREADSHEET_ID = 'YOUR_SPREADSHEET_ID';
   const FORM_ID = 'YOUR_FORM_ID';
   ```
   
2. **SPREADSHEET_ID 찾기**:
   - Google Sheets URL에서 찾기
   - 예: `https://docs.google.com/spreadsheets/d/[여기가_ID]/edit`
   
3. **FORM_ID 찾기**:
   - Google Forms URL에서 찾기
   - 예: `https://docs.google.com/forms/d/[여기가_ID]/edit`

### 4단계: 초기 설정 실행

1. Apps Script 편집기에서 `initializeSheets` 함수 선택
2. 실행 버튼 클릭
3. 권한 승인 (처음 실행 시)
4. 시트가 자동으로 생성되었는지 확인:
   - 참가자명단
   - 설문응답
   - 점수계산
   - 요약

### 5단계: 참가자 명단 입력

1. Google Sheets의 `참가자명단` 시트 열기
2. 다음 형식으로 참가자 정보 입력:
   ```
   이름    | 팀명  | 완료여부
   홍길동  | 팀A   | 
   김철수  | 팀A   |
   이영희  | 팀B   |
   ```

### 6단계: Google Forms 생성 (자동 생성)

**방법 1: 자동 생성 (권장)**
1. Apps Script 편집기에서 `createFormAutomatically` 함수 선택
2. 실행 버튼 클릭
3. 실행 로그에서 생성된 설문지 URL과 ID 확인
4. `Code.gs` 파일의 `FORM_ID`를 생성된 ID로 업데이트

**방법 2: 수동 생성**
1. Google Forms에서 새 설문지 생성
2. `설문지_템플릿_가이드.md` 파일 참고하여 질문 구성
3. 설문지 ID를 `Code.gs`의 `FORM_ID`에 입력

### 7단계: 트리거 설정

1. Apps Script 편집기에서 왼쪽 메뉴 `트리거` 클릭
2. `트리거 추가` 클릭
3. 설정:
   - 함수: `processFormResponse`
   - 이벤트 소스: 양식에서
   - 이벤트 유형: 양식 제출 시
4. 저장

### 8단계: 웹 앱 배포

1. Apps Script 편집기에서 `배포` > `웹 앱` 클릭
2. 설정:
   - 실행 대상: 나
   - 액세스 권한: 모든 사용자
3. 배포 후 URL 복사 (관리자 대시보드 접속용)

## 📊 사용 방법

### 참가자용

1. Google Forms 링크로 접속
2. 이름과 팀명을 드롭다운에서 선택
3. 본인 팀을 제외한 모든 팀에 대해:
   - 11개 항목 각각 1-10점으로 평가
   - MVP 점수 1-10점으로 평가
4. 제출

**참고**: 본인 팀 평가는 자동으로 제외되므로, 본인 팀 섹션은 건너뛰셔도 됩니다.

### 관리자용

1. 배포된 웹 앱 URL로 접속
2. 실시간으로 다음 정보 확인:
   - 참가자별 완료 현황
   - 팀별 최종 점수 및 순위
3. 자동 새로고침 옵션 사용 가능 (30초 간격)

## 🔧 점수 계산 로직

1. 각 참가자는 본인 팀을 제외한 모든 팀을 평가
2. 각 팀은 4개 카테고리, 10개 항목으로 평가됨 (총 100점 만점)
   - 창의성: 30점 (3개 항목 × 10점)
   - 구현 완성도: 30점 (3개 항목 × 10점)
   - 필요성: 30점 (3개 항목 × 10점)
   - 발표 전달력: 10점 (2개 항목 × 5점)
3. Google Forms에서는 모든 항목을 1-10점 척도로 입력받고, 시스템이 각 항목의 최대 점수에 맞춰 자동 변환
4. 각 팀이 받은 점수는 모든 평가자의 점수 합계
5. 최종 점수 = 총점수 ÷ 팀원 수
6. MVP 점수도 별도로 계산됨 (평균값)
7. 이를 통해 팀원 수가 다른 팀도 공정하게 평가됨

## 📝 채점 항목 구조

각 팀은 다음 4개 카테고리, 총 10개 항목으로 평가됩니다:

### 1. 창의성 (30점)
- 아이디어와 결과물의 창의성 (10점)
- 실행 가능성과 독창성 (10점)
- 차별성 (10점)

### 2. 구현 완성도 (30점)
- 필수 기능 구현 (10점)
- 오류 처리 (10점)
- 기술 이해도 (10점)

### 3. 필요성 (30점)
- 확장성 (10점)
- 주제 적합성 (10점)
- 상품성 (10점)

### 4. 발표 전달력 (10점)
- 발표 속도와 표현 (5점)
- 발표자료 구성 (5점)

**총 100점 만점**

각 항목은 Google Forms에서 1-10점 척도로 평가하며, 시스템이 자동으로 실제 점수로 변환합니다.

## ⚠️ 주의사항

1. **참가자 명단**: 설문 응답의 이름과 팀명은 명단과 정확히 일치해야 합니다
2. **권한 설정**: 처음 실행 시 Google 계정 권한 승인이 필요합니다
3. **트리거**: 설문지 응답이 자동으로 처리되려면 트리거가 설정되어 있어야 합니다
4. **데이터 형식**: 설문지의 질문 순서와 형식은 가이드를 따라야 합니다

## 🐛 문제 해결

### 설문 응답이 처리되지 않는 경우
- 트리거가 제대로 설정되었는지 확인
- `Code.gs`의 `FORM_ID`가 올바른지 확인
- Apps Script 실행 로그 확인 (보기 > 실행 로그)

### 점수가 계산되지 않는 경우
- `calculateScores` 함수를 수동으로 실행해보기
- `설문응답` 시트에 데이터가 있는지 확인

### 대시보드가 표시되지 않는 경우
- 웹 앱 배포가 완료되었는지 확인
- 배포 URL이 올바른지 확인
- 브라우저 콘솔에서 오류 확인
