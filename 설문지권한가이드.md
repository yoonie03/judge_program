# 설문지 소유권 및 권한 안내

## 설문지 소유권

### ✅ 네, 실행한 사람이 소유자가 됩니다!

`createFormAutomatically()` 함수를 실행하면:
- **생성된 설문지는 실행한 사람의 Google 계정에 생성됩니다**
- **실행한 사람이 자동으로 소유자가 됩니다**
- 실행한 사람의 Google Drive에 저장됩니다

### 예시

만약 "홍길동"이라는 Google 계정으로 로그인한 상태에서 `createFormAutomatically()` 함수를 실행하면:
- 설문지는 "홍길동"의 Google Drive에 생성됩니다
- "홍길동"이 설문지의 소유자가 됩니다
- 설문지 URL: `https://docs.google.com/forms/d/...` (홍길동 계정 소유)

## 권한 관리

### 설문지 공유하기

생성된 설문지를 다른 사람들과 공유하려면:

#### 방법 1: 편집 권한 공유
1. 생성된 설문지를 엽니다
2. 우측 상단의 **⚙️ 설정** 아이콘 클릭
3. **보내기** 탭에서 링크 공유 또는 이메일로 공유

#### 방법 2: 뷰어 권한 공유
1. 설문지 편집 화면에서 우측 상단의 **공유** 버튼 클릭
2. 공유할 사람의 이메일 입력
3. 권한 선택:
   - **편집자**: 설문지를 수정할 수 있음
   - **뷰어**: 설문지를 볼 수만 있음

### 설문 응답 권한

#### 응답자는 설문지만 작성하면 됩니다
- 설문지 링크만 있으면 누구나 응답할 수 있습니다
- 별도의 공유 권한이 필요 없습니다
- 응답은 자동으로 설문지 소유자의 계정에 저장됩니다

## 주의사항

### ⚠️ 중요한 점

1. **함수를 실행하는 계정**: 
   - 설문지는 함수를 실행한 계정에 생성됩니다
   - 다른 계정으로 실행하면 그 계정이 소유자가 됩니다

2. **SPREADSHEET_ID**:
   - 설문지 소유권과 별개로, `SPREADSHEET_ID`는 Google Sheets 파일의 ID입니다
   - 이 ID를 가진 Sheets 파일에 응답 데이터가 저장됩니다

3. **Apps Script 권한**:
   - Apps Script를 실행하는 계정이 설문지 소유자가 됩니다
   - 여러 계정에서 실행하면 각각 다른 설문지가 생성됩니다

## 권장 사항

### 1. 관리자 계정으로 실행
- 관리자 계정(공용 계정)으로 로그인한 상태에서 실행하는 것을 권장합니다
- 이렇게 하면 설문지가 관리자 계정에 생성되어 관리가 용이합니다

### 2. 공유 설정
- 설문지 생성 후 필요한 사람들에게만 편집 권한을 부여합니다
- 응답자는 설문지 링크만 있으면 응답할 수 있으므로 별도 권한이 필요 없습니다

### 3. 백업
- 생성된 설문지 URL과 ID를 안전한 곳에 기록해두세요
- 필요시 설문지를 복제하여 백업할 수 있습니다

## 설문지 소유권 확인 방법

1. 설문지를 엽니다
2. URL을 확인합니다:
   - `https://docs.google.com/forms/d/[ID]/edit`
   - 현재 로그인한 계정이 소유자입니다

3. 우측 상단 **공유** 버튼을 클릭하면 소유자 정보를 확인할 수 있습니다

## 요약

✅ **설문지를 만든 사람 = 소유자**
✅ **응답자는 별도 권한 불필요 (링크만 있으면 응답 가능)**
✅ **필요시 공유 기능으로 편집 권한 부여 가능**
